# 🏠 Housing Market Trends Analysis using Tableau

## Project Overview
This project analyzes a comprehensive dataset of residential properties to uncover the key factors influencing housing prices and sales trends. Using Tableau, the analysis explores how variables such as house age, years since renovation, number of bathrooms, bedrooms, and floors relate to overall market behavior. A key focus is assessing the impact of renovations on property value. The project delivers actionable insights through interactive dashboards and visual storytelling to support real estate analysts, marketing professionals, and decision-makers.
